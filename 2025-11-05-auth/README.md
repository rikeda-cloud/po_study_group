# 認証・認可学習用クイズ

このリポジトリは、OAuth 2.0 および OpenID Connect (OIDC) の認証・認可フローについて学習するためのクイズと解答、およびそれらを管理するためのPythonスクリプトを含んでいます。

## ディレクトリ構成

- `answer_md/`:
    - 認証・認可に関する説明が記述されたマークダウンファイルが含まれています。
    - 各ファイルには、重要なキーワードが`{}`で囲まれており、これがクイズの正解となります。

- `quiz_md/`:
    - `answer_md/`内のファイルから`{}`で囲まれたキーワードが`[____]`に置換された、虫食い形式のクイズファイルが格納されます。
    - `create_quiz.py`スクリプトによって生成されます。

## スクリプト

### `create_quiz.py`

- **説明**:
    - `answer_md/`ディレクトリ内のマークダウンファイルから、`{}`で囲まれたキーワードを`[____]`に置換し、虫食い形式のクイズファイルを`quiz_md/`ディレクトリに生成します。
- **使用方法**:
    ```bash
    python3 create_quiz.py
    ```

### `grade_quiz.py`

- **説明**:
    - `quiz_md/`ディレクトリ内のクイズファイルに記述された回答と、`answer_md/`ディレクトリ内の正解を比較し、ファイルごとの正答率を算出します。
    - `-v`または`--show-answers`オプションを付けることで、各問題の正解とあなたの回答を並べて表示し、正誤を確認できます。
- **使用方法**:
    - **採点のみ**:
        ```bash
        python3 grade_quiz.py
        ```
    - **詳細表示付きで採点**:
        ```bash
        python3 grade_quiz.py -v
        # または
        python3 grade_quiz.py --show-answers
        ```

## 学習方法

1.  `create_quiz.py`を実行して、`quiz_md/`ディレクトリにクイズファイルを生成します。(すでに生成済み)
2.  `quiz_md/`内のマークダウンファイルを開き、`[____]`の部分に適切なキーワードを記述して回答します。
3.  `grade_quiz.py`を実行して、あなたの回答を採点します。`-v`オプションを使って詳細を確認し、理解を深めましょう。
